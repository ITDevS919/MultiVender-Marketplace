# Marketplace Frontend

React frontend application built with Vite, TypeScript, and Tailwind CSS.

## Setup

1. Install dependencies:
```bash
npm install
```

## Development

Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`. API requests are configured to go to `http://localhost:5000/api` (make sure the backend server is running).

## Build

Build for production:
```bash
npm run build
```

The build output will be in the `build` directory.

## Type Checking

Run TypeScript type checking:
```bash
npm run type-check
```
